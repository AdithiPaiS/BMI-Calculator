# BMI Calculator

A BMI Calculator project developed using R programming to calculate and analyze Body Mass Index (BMI) using height and weight data.

## Features
- BMI calculation
- Data analysis and visualization
- Histogram, scatter plot, pie chart, and bar graph generation

## Technologies Used
- R Programming
- dplyr
- ggplot2
